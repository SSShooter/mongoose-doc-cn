### 操作
翻译 lib 文件夹和 docs 文件夹里的文件，           
docs 文件夹为教程，        
lib 文件夹为 api 文档
### 生成文档（jade->html）
```
// 预览前必须先生成文档
node website
```
### 本地服务
```
// 启动本地服务器预览文档
node static
```
